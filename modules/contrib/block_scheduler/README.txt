INTRODUCTION
------------
The Block expire module is condition plugin which adds configuartion
form in block with publish and expiry dates

 * For a full description of the module, visit the project page:
   https://www.drupal.org/sandbox/nishkris/2672728

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/2672728


INSTALLATION
------------

* Install as you would normally install a contributed Drupal module.
 See:https://www.drupal.org/documentation/install/modules-themes/modules-8
 for further information.


CONFIGURATION
-------------

The module has no menu or modifiable settings.
When enabled, the module will add a new visiblity settings in  configure block
